## About

DIY FlashAttention is an educational project that implements FlashAttention from scratch using Python and OpenAI Triton. Whether you're learning GPU programming or want to understand how modern LLMs achieve efficient attention computation, this project provides:

- **Clear explanations** of the core concepts
- **Working code** you can run and modify
- **Performance benchmarks** comparing different approaches
- **Step-by-step tutorials** covering everything from basics to advanced topics
