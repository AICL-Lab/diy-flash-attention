# Changelog Directory

This directory contains supporting changelog materials and archived release notes.

## Maintained Authorities

- English: [`../CHANGELOG.md`](../CHANGELOG.md)
- Chinese: [`./CHANGELOG.zh-CN.md`](./CHANGELOG.zh-CN.md)

Docs-site changelog pages should stay as thin routing pages that point back to these files instead of duplicating release notes.

## Archives

The `archive/` directory contains detailed changelogs for each release, organized by date and version.

## Viewing History

- **Latest English changes**: See [../CHANGELOG.md](../CHANGELOG.md)
- **Latest Chinese changes**: See [CHANGELOG.zh-CN.md](./CHANGELOG.zh-CN.md)
- **Detailed History**: Browse the [archive/](./archive/) directory
- **Git History**: Use `git log` for commit-level changes
